On snmp_container, get ip using "ip address"
On snmp_container2, get ip using "ip address"

On snmp_container, check if smpd process is running by running
"ps aux | grep snmpd"
if the output is only:
    root        43  0.0  0.0   3528  1640 pts/1    S+   08:49   0:00 grep --color=auto snmpd
then run:
    service snmpd start

On snmp_container2, ping snmp_container to get the ip
Using that ip, on snmp_container2, run
    snmpwalk -v 3 -u User -l authPriv -A AuthPassword -X PrivPassword -a SHA -x AES128 <IP>
